Show the code

Write it once. Cursor remembers.

Drop this in .cursor/rules/ and every prompt inherits it.

.cursor/rules/secure-defaults.md
# Secure defaults — applies to every file we generate.
When generating code:
- Validate every input that crosses a trust boundary.
- Never log secrets, tokens, or PII. Use the redact() helper.
- Prefer parameterized queries. Reject string concatenation in SQL.

When suggesting dependencies:
- Show the install command and the changelog of the last release.
- Flag any package with < 100 weekly downloads.

# That's it. Three rules, one file, every session.
