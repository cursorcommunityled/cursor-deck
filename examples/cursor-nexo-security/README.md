# Cursor × NEXO · Sesión 03 — Código Seguro con Cursor

Real deck used for the third session of the **Cursor × Fundación NEXO**
LATAM webinar series, April 2026. Topic: shipping secure code with Cursor
(Cursor Rules, prompt-engineered review, Semgrep MCP).

Speaker: Niccolò Mascaro · Cursor Ambassador · LATAM.

This folder is preserved verbatim as a reference for what a finished deck
looks like when this template is taken to production. **Do not edit it** —
copy the [`template/`](../../template/) folder instead.

## Run it

```bash
# from the repo root
npm run present:example
# → http://localhost:4321
```

Or `cd` into this folder and run any static server.
