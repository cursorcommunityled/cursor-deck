The loop

One epoch, many updates.

Forward pass → loss → backward → optimizer step. Repeat until validation says stop.

train.py
def train_one_epoch(model, loader, loss_fn, optimizer):
    model.train()
    for xb, yb in loader:
        optimizer.zero_grad()
        logits = model(xb)
        loss = loss_fn(logits, yb)
        loss.backward()
        optimizer.step()

Generalization
Remember this

Training minimizes loss on examples you have.
Deployment cares about examples you don't.

Hold out a validation set. Watch for overfitting. Match metrics to the real goal.
